Movsim Common
================

MovSim = **M**ulti-model **o**pen-source **v**ehicular-traffic **Sim**ulator.

http://www.movsim.org


Description
-----------

The `MovsimCommon` submodule provides some basic utilities used in the submodules `consumption`, `core` and `viewer`.
